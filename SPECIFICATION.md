# System Specification Document

## Introduction

* **Project Title:** TailorFit: AI Job Application Assistant
* **Domain:** Recruitment and Human Resources Technology. This domain covers digital platforms used to streamline hiring and assist job seekers in generating application materials.
* **Problem Statement:** Job seekers spend excessive time manually tailoring cover letters for specific roles. Generic applications fail to stand out, and manual tailoring is highly inefficient. TailorFit automates this process to improve application quality and save time.
* **Individual Scope:** To remain feasible for a single semester, the project is strictly constrained to text-to-text generation. The system will feature a single-page web interface and a lightweight Python backend. Complex features like PDF parsing and user authentication are excluded. AI generation relies on external APIs (like Google AI Studio) via a user-provided API key.
